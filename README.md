## How to Use

1. Import **NovaDx_AI_Lab_Automation_System_Final.json** into your n8n instance
2. Set up credentials — Google Gemini API, Google Sheets OAuth, Gmail OAuth, Twilio
3. Replace **YOUR_SHEETS_ID** with your Google Sheets file ID
4. Replace **YOUR_SANDBOX_NUMBER** with your Twilio sandbox number
5. Activate the workflow
